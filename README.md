# taskflow-web-app
TaskFlow is a responsive full-stack task management web application built with HTML, CSS, JavaScript, PHP, and MySQL. It allows users to create, organize, edit, and manage tasks through a clean and modern interface while storing data securely in a MySQL database.
## Features
-  Create new tasks
-  Edit existing tasks
-  Delete tasks
-  Mark tasks as completed or active
-  Organize tasks by category
-  Assign priority levels (High, Medium, Low)
-  Filter tasks by status and category
-  Responsive design for desktop and mobile
-  Toast notifications for user actions
-  Loading indicators
-  Secure backend using prepared SQL statements
-  AJAX-based CRUD operations without page reloads
